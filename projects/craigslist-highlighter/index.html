Craigslist Highlighter

Languages: Javascript
Summary: Helps find and highlight specific traits of apartments on craigslist
Source code: github [.tar.gz / .zip]

Details

It helps highlight things you’re looking for when searching for apartments or housing on craigslist. This script will follow each link on the search page and scan the individual listing’s body page for the keywords you enter. When it finds them, it will retrieve them and highlight them as positive or negative. It makes it easier to narrow attention down to the more interesting ones, but there’s still a fair way to go before I’m happy with it.
